C++ File Encrypter

C++

The Challenge

I needed a way to encrypt local files without relying on third-party tools. Most existing solutions were either overkill for small files or lacked transparency in how they handled keys.

The Solution

Built a command-line utility in C++ that reads a file, applies symmetric XOR-based encryption with a user-provided key, and writes the encrypted output. It handles binary and text files, validates input paths, and keeps the codebase under 200 lines.
